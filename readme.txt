steps to use bigDecimal(in link) converter 

1. first run the program 
2. find the localhost port no 
3. then put this link in browser  "http://localhost:8080/assignment/bigdecimal/"
4. append the number to be tested in the end of the link 
5. example "http://localhost:8080/assignment/bigdecimal/4595666.698"

steps to use bigDecimal(Postman) converter 

1. first run the program 
2. find the localhost port no 
3. then put this link in browser  "http://localhost:8080/assignment/bigdecimal"
4. use Postman to send the body 
5. in body use raw format enter the number
